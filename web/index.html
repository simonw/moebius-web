Moebius Inpainting

Moebius (0.22B) image-inpainting model, running in the browser with ONNX Runtime Web on WebGPU. Paint over a region to replace, then Run.

⚠️ The first run downloads ~1.27 GB of model weights from Hugging Face (the UNet alone is 907 MB). Your browser caches them, so later runs don't re-download. A WebGPU-capable browser (e.g. Chrome) is required.
