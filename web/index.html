Moebius Inpainting

Moebius (0.22B) image-inpainting model, running in the browser with ONNX Runtime Web on WebGPU. Paint over a region to replace, then Run.

⚠️ The first run downloads ~1.27 GB of model weights from Hugging Face (the UNet alone is 907 MB). They are stored in the browser's Cache Storage and reused on later visits — no re-download. A WebGPU-capable browser (recent Chrome or Safari) is required.
